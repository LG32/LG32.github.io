基础排序算法复习之一稳定性与时间复杂度

图1：

一：上图分析

1：算法稳定性：

在定义上来说排序前两个相等的数其在序列的前后位置顺序和排序后它们两个的前后位置顺序相同

既然排完序后是有序的，为什么要分稳点与不稳定：

例子：考试结束后， 班上的成绩单是按学号把各个同学的成绩列出来的，在对成绩进行排序后，稳定的排序方法可以使成绩相同的同学再按学号先后进行排序。

2：时间复杂度：

图2：时间复杂度

时间复杂度所耗费的时间是：

O(1) &lt; O(logn) &lt; O(n) &lt; O(nlogn) &lt; O(n2) &lt; O(n3) &lt;O(2n) &lt; O(n!) &lt;O(nn)

计算时间复杂度的方法：

大O推导法：

1.用常数1取代运行时间中的所有加法常数

2.在修改后的运行函数中，只保留最高阶项

3.如果最高阶项存在且不是1，则去除与这个项相乘的常数

例子：

1：线性阶：

for（int i = 0; i \&lt; n: i++){

//时间复杂度为1的程序

}

2：对数阶：

while(count \&lt; n){

    count = count \* 2;

    /\*时间复杂度为O(1)的程序\*/

}

因为每次count*2后，距离结束循环更近了。也就是说有多少个2 相乘后大于n，退出循环。

数学公式：2x = n –&gt; x = log2n

因此这个循环的时间复杂度为O(logn)

3：平方阶：

For(int i = 0; i \&lt; n; i++){

    For(int j = 0; j \&lt; n;  j++){

    //时间复杂度为1的程序

    }

}
